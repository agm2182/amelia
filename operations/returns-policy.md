# Cherri Returns & Exchange Policy

This document contains the official returns and exchange policy for Cherri underwear. It is referenced by customer service skills and can be used by any team member handling customer inquiries.

## Policy Overview

| Rule | Details |
|------|---------|
| **Return window** | 7 days after delivery (flexible with judgment) |
| **Refunds** | NO refunds - exchanges or store credit only |
| **Sale items** | YES - all items eligible for exchange |
| **Condition** | Items can be unopened, opened/tried on, or worn |
| **Order Protection** | Covers lost, stolen, damaged items via separate claim process |

## Core Rules

### 1. No Refunds Policy
Cherri does **not** offer refunds under any circumstances. Customers may receive:
- **Exchange** for a different size, color, or style
- **Store credit** if no suitable replacement is available

### 2. Return Window
The standard return window is **7 days from delivery date** (not order date).

**How to verify delivery date:**
- Check order details in Shopify for tracking information
- Use tracking number to confirm actual delivery date
- If tracking shows "delivered" more than 7 days ago, apply flexibility guidelines

### 3. All Items Eligible
Every item is eligible for exchange, including:
- Full-price items
- Sale items
- Bundles
- Limited editions

## Flexibility Guidelines

The 7-day rule is **not rigid**. Use judgment to approve exceptions for:

### Approve Exceptions For:
1. **Loyal/repeat customers** - Check order history; multiple previous orders = approve
2. **First-time customers** - Making a good first impression matters
3. **Valid reasons** - Customer was traveling, gift recipient opened late, delayed due to illness, etc.
4. **Polite customers** - Customers who are understanding and respectful

### When to Hold the Line:
1. **Significantly past window** (30+ days) with no good reason
2. **Repeat abuse** - Same customer requesting exceptions frequently
3. **Rude or demanding** customers (still be professional, but enforce policy)

### Example Scenarios:

| Scenario | Decision |
|----------|----------|
| 10 days past delivery, first-time buyer, apologetic | **Approve** |
| 8 days past delivery, 5 previous orders | **Approve** |
| 3 weeks past, customer was on vacation | **Approve** (with note) |
| 45 days past, no reason given, demanding tone | **Decline** politely |

## Item Condition Categories

### Unopened
- Still in original packaging
- Tags attached
- Never tried on

**Action:** Send replacement immediately, include return label for original

### Opened / Tried On
- Packaging opened
- May have been tried on once
- Still in new/like-new condition

**Action:** Send replacement, include return label for original

### Worn
- Customer has worn the item (not just tried on)
- Still wants a different size/style

**Action:** Send replacement, **no return needed** for worn item

## Stock Availability

### Shopify Inventory
Check Shopify for current stock levels. However, note:

**Important:** Shopify inventory may show "out of stock" but **extra stock may exist at the home office** for returns and replacements.

### Stock Check Process:
1. First, check Shopify inventory via MCP tools
2. If Shopify shows out of stock → Note that manual verification may be needed
3. Offer alternatives or store credit if truly unavailable

### When Stock is Unavailable:
- Offer similar style in same size
- Offer same style in different color
- Offer store credit for future purchase
- Check if item will be restocked soon

## Order Protection Claims

Order Protection is a third-party service that covers:
- **Lost packages** (tracking shows delivered but customer didn't receive)
- **Stolen packages**
- **Damaged items** (arrived broken, torn, defective)

### If Customer Has Order Protection:
Direct them to file a claim at **orderprotection.com** using their order confirmation email.

### If Customer Does NOT Have Order Protection:
Handle case-by-case:
- Loyal customers → Likely approve replacement
- First-time customers → Consider approving for goodwill
- Clearly damaged/defective → Approve (manufacturing issue)

### NOT Covered by Order Protection:
- Size exchanges (customer ordered wrong size)
- Style changes (customer wants different product)
- Color changes
- "Doesn't fit as expected"

These are **regular exchanges**, not Order Protection claims.

## Decision Tree

```
Customer requests exchange/return
    │
    ├─► CHECK: Order # provided?
    │   ├─► YES → Proceed to lookup
    │   └─► NO → Ask for order # or email used for order
    │
    ├─► LOOKUP ORDER in Shopify
    │   └─► Get: delivery date, items, Order Protection status
    │
    ├─► CHECK: Is this a damage/lost/stolen claim?
    │   ├─► YES + Has Order Protection → Direct to orderprotection.com
    │   ├─► YES + No Order Protection → Case-by-case (loyal customer?)
    │   └─► NO → Continue to size/style exchange
    │
    ├─► CHECK: Within 7-day window? (from delivery date)
    │   ├─► NO → Check for exceptions:
    │   │       • Loyal/repeat customer?
    │   │       • First-time customer?
    │   │       • Valid reason provided?
    │   │       • Polite and understanding?
    │   │       ├─► Exception applies → Approve anyway
    │   │       └─► No exception → Decline (past window)
    │   └─► YES → Continue
    │
    ├─► DETERMINE: Item condition
    │   ├─► Unopened → Send replacement + return label
    │   ├─► Opened/tried on → Send replacement + return label
    │   └─► Worn → Send replacement, NO return needed
    │
    └─► CHECK: Replacement in stock?
        ├─► In Shopify → Confirm & process
        ├─► Not in Shopify → Note: may have home office stock
        └─► Truly unavailable → Offer alternatives or store credit
```

## Exchange Process

### Standard Exchange Flow:
1. Customer requests exchange (via returns portal or email)
2. Look up order in Shopify
3. Verify eligibility (window + condition)
4. Check replacement stock
5. Send confirmation email with next steps
6. Generate return label from Shopify order page (if needed)
7. Ship replacement item

### Returns Portal vs Direct Email:
- **Returns Portal:** Usually includes order #, size preference, reason
- **Direct Email:** May need to ask for order # or size preference

## Sizing Guidance

When customers ask what size to exchange for:

### Questions to Consider:
1. What size did they originally order?
2. What's the fit issue? (too tight, too loose, cups too small, band too big, etc.)
3. What's their preference? (prefer tighter fit? looser fit?)

### General Guidance:
- "Too tight in band" → Go up a band size
- "Too loose in band" → Go down a band size
- "Cups too small" → Go up a cup size (or suggest sister size)
- "Cups too big" → Go down a cup size
- "Overall too small" → Go up one full size
- "Overall too big" → Go down one full size

### Sister Sizes (for bras):
Same cup volume, different band:
- 32C = 34B = 36A
- 32D = 34C = 36B
- etc.

## Important Notes

1. **Always be warm and helpful** - Even when declining, maintain Cherri's friendly brand voice
2. **Use judgment** - These guidelines aren't rigid rules
3. **Document exceptions** - Note why an exception was made for future reference
4. **Escalate if unsure** - When in doubt, ask a team lead
5. **Shipping labels** - Generated from Shopify order page after approval
